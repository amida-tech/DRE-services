###Prerequisites

- Node.js (v0.10+) and NPM
- Grunt.js
- MongoDB
- Redis

#### prepare
```
# you need Node.js and Grunt.js installed
# and MongoDB + Redis runnning

npm install -g grunt-cli

#then
npm install
```

To run, use `node server`

#### Grunt commands:

`grunt` - To run Server Side tests
